# 🍕 The Pizza Edition | Unblocked Games

Welcome to **The Pizza Edition**, the ultimate web gaming portal featuring a massive collection of fast, free, and fun unblocked games! Whether you're a fan of retro arcade classics, modern 3D adventures, or casual idle clickers, there's a slice of fun here for everyone.

## ✨ Features

- **Massive Game Library**: Discover over 150+ games spread across multiple distinct categories.
- **Advanced 3D Experiences**: Play immersive 3D web games like *Red vs Blue 3D* (featuring full capture-the-flag mechanics and ground AI) and our *Red Dead Redemption 2* inspired adventure, complete with functional mini-maps and interactive NPC quests!
- **Cross-Platform & Mobile Friendly**: Robust touchscreen controls and dynamic layouts ensure our games play perfectly on mobile devices, tablets, and desktops alike.
- **P2P Multiplayer**: Connect and play with friends using our seamless WebRTC and PeerJS multiplayer integration, which includes robust manual bypass capabilities for restrictive networks.
- **🤖 Dedicated AI Gaming Chatbot**: Need game recommendations, walkthroughs, or tips? Chat with our integrated AI assistant—custom-locked and focused entirely on all things gaming!
- **Global Leaderboards**: Prove your skills and compete for the top score across various games in the 🏆 Leaderboard.

## 🎮 Game Categories

- **Modern Games**: Contemporary web hits including *Tower Defence*, *Fallout: Terminal*, *Zombie Game*, and *Space Flight Simulator*.
- **3D Games**: Dive into rich 3D worlds directly in your browser, complete with responsive movement controls and advanced logic.
- **Classic Games**: Retro goodness featuring dozens of timeless classics including *Pac-Man*, *Asteroids*, *Space Invaders*, *Tetris*, *Snake*, and *Galaga*.
- **2D & Clicker Games**: Need a quick distraction? Enjoy an extensive array of 2D puzzles and satisfying idle clicker games.

## 🛠️ Built With

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Styling**: Custom CSS featuring modern gradients, UI animations, and clean typography (Google's Outfit font).
- **Multiplayer**: PeerJS / WebRTC for fast peer-to-peer data connections.
- **Graphics & Rendering**: Vanilla Canvas API and WebGL constructs for performant 2D/3D in-browser gaming.

## 🚀 Getting Started

To run **The Pizza Edition** locally on your machine:

1. Clone or download the repository to your local drive.
2. For the best experience (and to prevent CORS issues with local assets), run a local web server in the root directory:
   - **Using Python:** `python -m http.server 8000`
   - **Using Node.js:** `npx serve .`
   - **Using VS Code:** Install and use the *Live Server* extension.
3. Open your browser and navigate to `http://localhost:8000` (or the port provided by your server).
4. Start gaming!

To deploy **The Pizza Edition** to the your thumb drive:

1. Run `npm run deploy` to deploy the project.
